## 📌 Project Overview

This project is a **modern responsive website layout** built using **HTML and CSS**.
It focuses on creating a visually appealing **Hero Section** with proper alignment, responsiveness, and smooth animations.

The design ensures that the hero section fits perfectly within the screen using **viewport units (`vh`, `vw`)**.
